Introduction to Proportions in Statistics

In AP Statistics, we often need to analyze proportions - the fraction of a population or sample with a particular characteristic. This guide covers:

  • One-proportion confidence intervals
  • Two-proportion confidence intervals
  • One-proportion hypothesis tests
  • Two-proportion hypothesis tests

Each method serves a specific purpose in statistical analysis, and understanding when to use each approach is crucial for success in AP Statistics.

One-Proportion Confidence Intervals

A one-proportion confidence interval estimates the true population proportion (p) based on a single sample proportion.

When to Use

Use a one-proportion confidence interval when you want to estimate an unknown population proportion based on a single sample.

Conditions for Validity

  • Random Sample: The data comes from a random sample or randomized experiment
  • Independence: Observations are independent (sample size ≤ 10% of population or sampling with replacement)
  • Large Sample: Both np ≥ 10 and n(1-p) ≥ 10, where p̂ is the sample proportion

Formula

p̂ ± z* × √[p̂(1-p̂)/n]

Where:
p̂ = x/n (sample proportion)
x = number of successes
n = sample size
z* = critical value for desired confidence level (e.g., 1.96 for 95% confidence)

Example

Problem: A random sample of 120 AP Statistics students found that 84 passed the AP exam. Construct a 95% confidence interval for the true proportion of all AP Statistics students who pass the exam.

Solution:

  1. Calculate the sample proportion: p̂ = 84/120 = 0.7
  2. Check conditions:
    • Random sample ✓
    • Independence: 120 is less than 10% of all AP Statistics students ✓
    • Large sample: np̂ = 120(0.7) = 84 ≥ 10 and n(1-p̂) = 120(0.3) = 36 ≥ 10 ✓
  3. Critical value for 95% confidence: z* = 1.96
  4. Standard error: √[p̂(1-p̂)/n] = √[0.7(0.3)/120] = √[0.21/120] = √0.00175 = 0.0418
  5. Margin of error: 1.96 × 0.0418 = 0.082
  6. Confidence interval: 0.7 ± 0.082 = (0.618, 0.782)

Interpretation: We are 95% confident that the true proportion of all AP Statistics students who pass the exam is between 0.618 and 0.782 (or between 61.8% and 78.2%).

Two-Proportion Confidence Intervals

A two-proportion confidence interval estimates the difference between two population proportions (p₁ - p₂) based on two sample proportions.

When to Use

Use a two-proportion confidence interval when comparing proportions from two different groups or populations.

Conditions for Validity

  • Random Samples: Both samples are random or from randomized experiments
  • Independence: Samples are independent of each other
  • Large Samples: For both samples, np ≥ 10 and n(1-p) ≥ 10

Formula

(p̂₁ - p̂₂) ± z* × √[p̂₁(1-p̂₁)/n₁ + p̂₂(1-p̂₂)/n₂]

Where:
p̂₁ = x₁/n₁ (first sample proportion)
p̂₂ = x₂/n₂ (second sample proportion)
n₁, n₂ = sample sizes
z* = critical value for desired confidence level

Example

Problem: A study compared two test preparation methods for the AP Statistics exam. In group A, 78 out of 100 students passed, while in group B, 65 out of 95 students passed. Construct a 90% confidence interval for the difference in passing proportions between methods A and B.

Solution:

  1. Calculate sample proportions:
    • p̂₁ = 78/100 = 0.78 (group A)
    • p̂₂ = 65/95 = 0.684 (group B)
  2. Check conditions:
    • Random samples ✓
    • Independence: Groups A and B are separate ✓
    • Large samples: All np and n(1-p) values ≥ 10 ✓
  3. Critical value for 90% confidence: z* = 1.645
  4. Standard error: √[0.78(0.22)/100 + 0.684(0.316)/95] = √[0.00172 + 0.00228] = √0.004 = 0.0632
  5. Margin of error: 1.645 × 0.0632 = 0.104
  6. Confidence interval: (0.78 - 0.684) ± 0.104 = 0.096 ± 0.104 = (-0.008, 0.2)

Interpretation: We are 90% confident that the difference in passing proportions between methods A and B (p₁ - p₂) is between -0.008 and 0.2. Since this interval contains 0, we cannot be confident at the 90% level that there is a real difference between the methods.

One-Proportion Hypothesis Testing

A one-proportion hypothesis test determines whether a sample proportion provides evidence against a claimed value for the population proportion.

When to Use

Use a one-proportion hypothesis test when you want to test a claim about a single population proportion.

Hypotheses

Typically written as:

  • H₀: p = p₀ (null hypothesis with claimed value)
  • H₁: p < p₀ or p > p₀ or p ≠ p₀ (alternative hypothesis)

Conditions for Validity

  • Random Sample: The data comes from a random sample or randomized experiment
  • Independence: Observations are independent
  • Large Sample: Both np₀ ≥ 10 and n(1-p₀) ≥ 10, using the null hypothesis value p₀

Test Statistic

z = (p̂ - p₀) / √[p₀(1-p₀)/n]

Where:
p̂ = x/n (sample proportion)
p₀ = claimed population proportion (null hypothesis value)
n = sample size

P-value

The p-value depends on the alternative hypothesis:

  • H₁: p < p₀: P(Z ≤ z)
  • H₁: p > p₀: P(Z ≥ z)
  • H₁: p ≠ p₀: 2 × P(Z ≥ |z|)

Example

Problem: A company claims that 60% of students who use their AP Statistics review book pass the exam. A statistics teacher is skeptical and believes the rate is lower. In a random sample of 90 students who used the book, 45 passed the exam. Test the company's claim at α = 0.05.

Solution:

  1. State hypotheses:
    • H₀: p = 0.6
    • H₁: p < 0.6 (one-sided test as the teacher believes the rate is lower)
  2. Calculate sample proportion: p̂ = 45/90 = 0.5
  3. Check conditions:
    • Random sample ✓
    • Independence ✓
    • Large sample: np₀ = 90(0.6) = 54 ≥ 10 and n(1-p₀) = 90(0.4) = 36 ≥ 10 ✓
  4. Calculate test statistic: z = (0.5 - 0.6) / √[0.6(0.4)/90] = -0.1 / √0.0027 = -0.1 / 0.052 = -1.92
  5. Calculate p-value: For H₁: p < 0.6, p-value = P(Z ≤ -1.92) = 0.0274
  6. Decision: Since p-value = 0.0274 < α = 0.05, reject H₀

Conclusion: There is sufficient evidence to suggest that the passing rate is less than the company's claim of 60%.

Two-Proportion Hypothesis Testing

A two-proportion hypothesis test determines whether there is a significant difference between two population proportions.

When to Use

Use a two-proportion hypothesis test when comparing proportions from two different groups or populations.

Hypotheses

Typically written as:

  • H₀: p₁ = p₂ (or p₁ - p₂ = 0)
  • H₁: p₁ < p₂ or p₁ > p₂ or p₁ ≠ p₂

Conditions for Validity

  • Random Samples: Both samples are random or from randomized experiments
  • Independence: Samples are independent of each other
  • Large Samples: For both samples, np̂ ≥ 10 and n(1-p̂) ≥ 10 (using pooled proportion for checking)

Pooled Proportion

p̂pooled = (x₁ + x₂) / (n₁ + n₂)
Where x₁, x₂ are successes in each sample

Test Statistic

z = (p̂₁ - p̂₂) / √[p̂pooled(1-p̂pooled)(1/n₁ + 1/n₂)]

Where:
p̂₁ = x₁/n₁ (first sample proportion)
p̂₂ = x₂/n₂ (second sample proportion)

P-value

The p-value depends on the alternative hypothesis:

  • H₁: p₁ < p₂: P(Z ≤ z)
  • H₁: p₁ > p₂: P(Z ≥ z)
  • H₁: p₁ ≠ p₂: 2 × P(Z ≥ |z|)

Example

Problem: A researcher wants to know if there's a difference in passing rates between students who used online review materials vs. traditional textbooks for AP Statistics. Among 120 students who used online materials, 90 passed. Among 100 students who used textbooks, 65 passed. Test at α = 0.05.

Solution:

  1. State hypotheses:
    • H₀: p₁ = p₂ (no difference in passing rates)
    • H₁: p₁ ≠ p₂ (there is a difference in passing rates)
  2. Calculate sample proportions:
    • p̂₁ = 90/120 = 0.75 (online)
    • p̂₂ = 65/100 = 0.65 (textbook)
  3. Calculate pooled proportion:
    p̂pooled = (90 + 65) / (120 + 100) = 155/220 = 0.705
  4. Check conditions:
    • Random samples ✓
    • Independence ✓
    • Large samples: All np and n(1-p) ≥ 10 using pooled proportion ✓
  5. Calculate test statistic:
    z = (0.75 - 0.65) / √[0.705(0.295)(1/120 + 1/100)]
    = 0.1 / √[0.208(0.0083 + 0.01)]
    = 0.1 / √[0.208 × 0.0183]
    = 0.1 / √0.0038
    = 0.1 / 0.0617
    = 1.62
  6. Calculate p-value: For H₁: p₁ ≠ p₂, p-value = 2 × P(Z ≥ 1.62) = 2 × 0.0526 = 0.1052
  7. Decision: Since p-value = 0.1052 > α = 0.05, fail to reject H₀

Conclusion: There is not sufficient evidence to conclude that there is a difference in passing rates between online review materials and traditional textbooks.

Comparison: One vs. Two Proportion Methods

Confidence Intervals

One-Proportion CI

  • Purpose: Estimate a single population proportion
  • Data: One sample
  • Formula: p̂ ± z* × √[p̂(1-p̂)/n]
  • Example Use: Estimating the proportion of all AP Statistics students who will pass the exam

Two-Proportion CI

  • Purpose: Estimate the difference between two population proportions
  • Data: Two independent samples
  • Formula: (p̂₁ - p̂₂) ± z* × √[p̂₁(1-p̂₁)/n₁ + p̂₂(1-p̂₂)/n₂]
  • Example Use: Comparing pass rates between two different teaching methods

Hypothesis Tests

One-Proportion Test

  • Purpose: Test a claim about a single population proportion
  • Null Hypothesis: p = p₀
  • Test Statistic: z = (p̂ - p₀) / √[p₀(1-p₀)/n]
  • Example Use: Testing if a passing rate differs from a claimed value

Two-Proportion Test

  • Purpose: Test if two population proportions differ
  • Null Hypothesis: p₁ = p₂
  • Test Statistic: z = (p̂₁ - p̂₂) / √[p̂pooled(1-p̂pooled)(1/n₁ + 1/n₂)]
  • Example Use: Testing if two teaching methods produce different passing rates

Key Differences Table

Feature One Proportion Two Proportion
Number of samples One Two
What we're estimating/testing A single proportion p Difference between proportions (p₁ - p₂)
Null hypothesis p = p₀ (specified value) p₁ = p₂ (no difference)
Standard error formula √[p₀(1-p₀)/n] for tests
√[p̂(1-p̂)/n] for CIs		 √[p̂pooled(1-p̂pooled)(1/n₁ + 1/n₂)] for tests
√[p̂₁(1-p̂₁)/n₁ + p̂₂(1-p̂₂)/n₂] for CIs
Pooled proportion needed? No Yes, for hypothesis tests only

AP Exam Tip

On the AP exam, make sure to identify whether a problem involves one or two proportions before selecting a method. Look for keywords like "difference," "compare," or "two groups" that suggest a two-proportion method is needed.
